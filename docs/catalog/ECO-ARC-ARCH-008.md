# ECO-ARC-ARCH-008

**Name:** Hot storage used for cold data

**Category:** ARC

**Family:** ARCH

**Primary layer:** `architecture`

**System layers:** `architecture`

## Description

Keeping cold data in hot tiers wastes storage spend and energy.

## Impact

- **confidence:** 0.7
- **notes:** Often large savings.
- **type:** storage

## Detection

- **languages:**
  - infra
- **method:** trace

## Remediation

- **guidance:** Introduce lifecycle policies and cold tiers.
- **tradeoffs:** Retrieval latency and costs.
