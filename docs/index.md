# Eco Rules Catalog

The Eco Rules Catalog is a SITM-owned standard for identifying, describing, and organizing software inefficiency.

It provides:

- a shared language for waste
- a canonical rule model
- a governed taxonomy and registry
- versioned release artifacts
- a foundation for tools, education, and certification

## Why this exists

Software waste is often invisible until it becomes cost, fragility, delay, or environmental impact.

The Eco Rules Catalog exists to make that waste visible and actionable.

Because waste is a bug.
